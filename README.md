## Features

- **Revenue-verified leaderboard**
- **Solo founder profiles**
- **Public revenue tracking**
- **Competitive rankings**
- **Community of ambitious builders**

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions, Storage)
- **AI:** Claude / Gemini via API
- **Hosting:** Lovable
- **Build Tool:** Vite

## Getting Started

```bash
# Clone the repo
git clone https://github.com/LazyUnicornFounder/solo-unicorn-league.git
cd solo-unicorn-league

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your Supabase credentials to .env

# Start the dev server
npm run dev
```
